# KTH Course Catalog Filter

This Streamlit app allows filtering of KTH courses by semester, period (P1–P4), and whether the course has a final exam ("Tentamen").

## Features

- Scrapes KTH course catalog search results and detail pages
- Caches results for faster repeat runs
- Dynamic UI with filters and CSV export

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
